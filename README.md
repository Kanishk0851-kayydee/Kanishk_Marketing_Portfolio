# Kanishk Dawar | Brand & Growth Marketing Dashboard

A sleek, dark-mode, and highly vibrant portfolio dashboard designed to showcase cross-functional marketing achievements, campaign execution frameworks, and visual media galleries. Built using React, Next.js, and Tailwind CSS, and optimized for instant hosting on Vercel.

## 📁 Repository Structure

Your GitHub repository must follow this layout for the configurations and Tailwind styling to build successfully:

```text
├── app/
│   ├── globals.css      # Custom styles and Tailwind directives
│   ├── layout.js        # Next.js HTML core layout wrapper
│   └── page.js          # The main interactive portfolio dashboard code
├── public/
│   └── KanishkDawar_Brand and Growth Marketing_2026.pdf
├── package.json         # Project metadata and required dependencies
├── postcss.config.js    # PostCSS styling compilation rules
└── tailwind.config.js   # Tailwind CSS paths and style configuration
